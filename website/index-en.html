Omi
Open and modern framework for building user interfaces.
Github

  • Scoped CSS

    Reusable component is composed of HTML, Scoped CSS and JS. Don't worry about the CSS of component will affect other components.

  • Super Tiny Size

    7 kb (gzip)
    Although the file size is small, but it has very good compatibility! Compatible with IE8

  • Update with Free Style

    Each component has a update method that allows you to choose the best time to update.

  • Any Template Engine

    Developers can override the Omi.template method to use any template engine.

  • Object-Oriented Programming

    Omi uses a completely object-oriented approach to build Web programs.

  • Support ES6+ and ES5

    Omi provides two development programs for ES6+ and ES5. You can choose the way you like.

  • npm install omi-cli -g
  • omi init your_project_name
  • cd your_project_name
  • npm run dev
  • npm run dist
