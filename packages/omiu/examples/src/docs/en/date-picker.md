## DatePicker 

Date picker.

## Usage

```js
<o-date-picker
  selectedDate={this.selectedDate}
  onSelect={this.onSelect}
  show={this.showPicker}
  onToggle={this.onToggle}>
</o-date-picker>
```

## API

### Props

|  **Name**  | **Type**        | **Defaults**  | **Details**  |
| ------------- |:-------------:|:-----:|:-------------:|

| selectedDate  | string| -- |  |
| onToggle | function   |    | |
